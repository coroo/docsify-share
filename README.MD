# docsify-share

## Install

1. Configure docsify-share:

    ```html
    <script>
        window.$docsify = {
            share: {
                reddit: true,
                linkedin: true,
                facebook: true,
                twitter: true,
                whatsapp: true,
                telegram: true,
            }
        };
    </script>
    ```

2. Insert script into docsify document:

    ```html
    <script src="//unpkg.com/docsify-share/build/index.min.js"></script>
    ```

## Usage

It will automatically show you a share button link.
